# Redis 搭建模式

Redis 的搭建方式：

- 单机
- 主从
- 哨兵
- 集群

单机，只要一台Redis服务器，挂了就无法工作了

主从，是备份关系， 数据也会同步到从库，还可以读写分离

哨兵：master挂了，哨兵就行选举，选出新的master，作用是监控主从，主从切换

集群：高可用，分散请求。目的是将数据分片存储，节省内存。
